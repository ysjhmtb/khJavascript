jQuery 개요

jQuery 란~?

복잡한 javascript 식을 대신하여 손쉽게 사용하기 위해 나온 javascript 기반 (경량)라이브러리

jquery의 장점
1. DOM과 객체들의 처리가 쉽다.(선택자, 문법)
2. ajax, 이벤트처리와 같은 기능을 다양하게 제공한다.
3. 플러그인을 사용하여 UI를 구현, 차트, 슬라이드, 테이블과 같은 기능을 손쉽게 구현 가능하다.

jquery 연동 방법

1. script - jquery 선언

2. script - src(link - cdn)(web, local)

jQuery(document).ready();

페이지가 로드 됬을때 호출되는 이벤트 핸들러
window.onload, body-onload 같은 개념의 메소드
jquery 선언 한 ready 메소드의 경우 중복 선언이 가능 함

jQuery 버전

1.x : 구버전 브라우저를 지원(IE 6,7,8) => jquery compact 3.0 2.x : 신버전 브라우저를 지원 + 코드 경량화 => jquery 3.0
