속성 선택자

해당 엘리먼트의 속성 값을 선택자에 이용한다.
태그 선택 후 속성 선택자를 추가해서 사용한다.

요소[속성명] : 해당 속성이 존재하는 요소만 선택
요소[속성 = 값] : 속성 값이 일치하는 요소만 선택
요소[속성 ~= 값] : 특정 단어를 포함하는 요소만 선택(띄어쓰기로 속성값을 구분)
요소[속성 *= 값] : 속성값에 해당 값을 포함하는 요소만 선택
요소[속성 ^= 값] : 속성값이 해당값으로 시작 하는 요소만 선택
요소[속성 $= 값] : 속성 값이 해당 값으로 끝나는 요소만 선택
